# Dotfiles (Arch Linux + Hyprland)

My personal Arch Linux Desktop

## Includes
- Hyprland (Wayland)
- Waybar
- Rofi
- Kitty
- AGS
- Wallust
- GTK / QT theming
- Full package lists (pacman + AUR)

## Install (Fresh Arch)
```bash
git clone https://github.com/Kohzmik/dotfiles
cd dotfiles
chmod +x scripts/*.sh
./scripts/install.sh
